Developed modular Retrieval-Augmented Generation system using Qwen3 model for automated textbook question answering with extensible document processing pipeline.

## Technologies Used

* Python
* LangChain
* Qwen3-0.6B
* ChromaDB
* Sentence Transformers
* Streamlit

## Key Features

* Semantic search with vector database
* Interchangeable components
* Document processing pipeline
* Automated Q&A system
